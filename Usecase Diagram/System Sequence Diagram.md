


# 基本功能

 1. 登录
 2. 注册
 3. 发布任务
 4. 修改密码/邮箱
 5. 查找任务
  
  
学生可以在我们的网站上获取相关任务，完成任务获得积分，再将积分提取为现金，从而达到挣闲钱的目的。
